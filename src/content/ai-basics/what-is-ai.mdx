# 🤖 AI란 무엇인가요?

인공지능(Artificial Intelligence, AI)은 인간의 지능을 모방하여 학습하고, 추론하고, 문제를 해결할 수 있는 컴퓨터 시스템을 말해요.

## 💡 AI의 핵심 능력

AI는 크게 네 가지 능력을 갖추고 있어요:

| 능력 | 설명 | 예시 |
|------|------|------|
| **인식** | 이미지, 음성, 텍스트를 이해해요 | 얼굴 인식, 음성 인식 |
| **학습** | 데이터에서 패턴을 찾아요 | 스팸 메일 분류 |
| **추론** | 논리적 판단을 내려요 | 의료 진단 보조 |
| **생성** | 새로운 콘텐츠를 만들어요 | 글쓰기, 그림 그리기 |

## 🏠 일상 속 AI

우리는 이미 매일 AI를 사용하고 있어요:

- **음성 비서** — 시리, 빅스비, 알렉사에게 날씨를 물어보는 것
- **추천 시스템** — 넷플릭스가 "이 영화 좋아할 걸요?" 하는 것
- **번역** — 파파고, 구글 번역으로 외국어를 이해하는 것
- **자동완성** — 이메일이나 검색어를 미리 예측하는 것
- **네비게이션** — 실시간 교통 상황을 분석해 최적 경로를 안내하는 것

## 🎯 AI의 세 가지 수준

AI는 능력에 따라 세 단계로 나눌 수 있어요:

### 1. 약한 AI (Narrow AI)
특정 작업 하나를 잘 수행하는 AI예요. 현재 우리가 사용하는 대부분의 AI가 여기에 해당해요.
- 체스를 두는 AI, 번역하는 AI, 얼굴을 인식하는 AI

### 2. 일반 AI (AGI)
인간처럼 다양한 분야에서 범용적으로 사고할 수 있는 AI예요. 아직 실현되지 않았지만, 많은 연구가 진행 중이에요.

### 3. 초인공지능 (ASI)
인간의 지능을 뛰어넘는 가상의 AI예요. 현재로서는 이론적 개념이에요.

## 🔑 AI vs 전통 프로그래밍

전통적인 프로그래밍과 AI의 가장 큰 차이점은 **학습**이에요:

```
전통 프로그래밍: 규칙을 사람이 직접 작성
  입력 + 규칙 → 출력

AI/머신러닝: 데이터로부터 규칙을 학습
  입력 + 출력 → 규칙 (자동 학습)
```

예를 들어 "고양이 사진을 판별하는 프로그램"을 만든다고 해볼게요:
- **전통 방식**: "귀가 뾰족하고, 수염이 있고..." 하고 규칙을 일일이 작성해요
- **AI 방식**: 수천 장의 고양이 사진을 보여주고, AI가 스스로 특징을 학습해요

## 📌 핵심 정리

- AI는 **인간의 지능을 모방**하는 컴퓨터 시스템이에요
- 현재 AI는 대부분 **특정 작업에 특화된 약한 AI**예요
- AI의 핵심은 **데이터로부터 스스로 학습**하는 것이에요
- 우리는 이미 일상에서 AI를 널리 활용하고 있어요
