# 📊 머신러닝 쉽게 이해하기

머신러닝(Machine Learning)은 컴퓨터가 명시적으로 프로그래밍하지 않아도 데이터로부터 패턴을 배우는 기술이에요.

## 🍎 비유로 이해하기

아이에게 "사과"를 가르치는 상황을 생각해 볼게요:

1. 빨간 사과, 초록 사과, 노란 사과를 여러 개 보여줘요
2. 매번 "이건 사과야"라고 알려줘요
3. 아이는 점점 새로운 사과도 알아보게 돼요

머신러닝도 똑같아요! 많은 데이터(사과 사진)와 정답(사과라는 라벨)을 주면, 컴퓨터가 스스로 규칙을 찾아내요.

## 🎓 머신러닝의 세 가지 학습 방법

### 1. 지도학습 (Supervised Learning)
정답이 있는 데이터로 학습하는 방식이에요.

- **분류(Classification)**: 이메일이 스팸인지 아닌지 구분
- **회귀(Regression)**: 집값을 예측

```
데이터: [면적: 30평, 역세권, 신축] → 정답: 8억
데이터: [면적: 20평, 비역세권, 구축] → 정답: 4억
→ 새 집의 가격을 예측!
```

### 2. 비지도학습 (Unsupervised Learning)
정답 없이 데이터의 패턴을 스스로 찾는 방식이에요.

- **군집화(Clustering)**: 비슷한 고객을 그룹으로 묶기
- **차원 축소**: 복잡한 데이터를 단순하게 만들기

### 3. 강화학습 (Reinforcement Learning)
시행착오를 통해 보상을 최대화하는 방식이에요.

- 게임 AI가 이기면 보상, 지면 패널티를 받으며 전략을 개선
- AlphaGo가 바둑을 학습한 방식이 바로 강화학습이에요

## 🔄 머신러닝 학습 과정

머신러닝 모델이 학습하는 과정은 다음과 같아요:

1. **데이터 수집** — 충분한 양의 학습 데이터를 모아요
2. **데이터 전처리** — 결측치 제거, 정규화 등 데이터를 정리해요
3. **모델 선택** — 문제에 맞는 알고리즘을 선택해요
4. **학습(Training)** — 데이터를 모델에 넣고 패턴을 학습시켜요
5. **평가(Evaluation)** — 새로운 데이터로 성능을 확인해요
6. **개선** — 부족한 부분을 찾아 반복적으로 개선해요

## ⚠️ 자주 하는 오해

| 오해 | 현실 |
|------|------|
| "ML은 만능이다" | 좋은 데이터가 없으면 좋은 모델도 없어요 |
| "데이터가 많을수록 좋다" | 양보다 **품질**이 더 중요해요 |
| "ML = AI 전체" | ML은 AI의 한 분야예요 |
| "한 번 학습하면 끝" | 환경이 바뀌면 재학습이 필요해요 |

## 🛠️ 실생활 적용 사례

- **넷플릭스**: 시청 기록을 분석해 취향에 맞는 콘텐츠를 추천해요
- **은행**: 거래 패턴을 학습해 이상 거래(사기)를 탐지해요
- **병원**: X-ray 이미지를 분석해 질병을 조기에 발견해요
- **자율주행**: 도로 상황을 실시간으로 인식하고 판단해요

## 📌 핵심 정리

- 머신러닝은 **데이터에서 패턴을 찾아 예측**하는 기술이에요
- 학습 방식은 **지도학습**, **비지도학습**, **강화학습** 세 가지가 있어요
- **좋은 데이터**가 좋은 모델의 핵심이에요
