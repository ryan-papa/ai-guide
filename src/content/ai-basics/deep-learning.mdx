# 🧠 딥러닝은 어떻게 배우나요?

딥러닝(Deep Learning)은 인간의 뇌를 모방한 **인공 신경망**을 여러 층으로 쌓아, 복잡한 패턴을 학습하는 머신러닝의 한 분야예요.

## 🏗️ 신경망의 구조

인공 신경망은 뇌의 뉴런을 본떠 만들었어요:

```
입력층        은닉층(여러 개)      출력층
[픽셀값]  →  [특징 추출]      →  [결과]
[색상]    →  [패턴 인식]      →  [고양이? 강아지?]
[형태]    →  [고수준 이해]    →
```

- **입력층**: 데이터를 받아들여요 (이미지의 픽셀, 텍스트의 단어 등)
- **은닉층**: 데이터의 특징을 추출하고 변환해요 (이 층이 "깊어서" 딥러닝!)
- **출력층**: 최종 결과를 내놓아요

## 🎯 딥러닝이 "깊은" 이유

머신러닝과 딥러닝의 차이는 **층의 깊이**에 있어요:

| 구분 | 머신러닝 | 딥러닝 |
|------|---------|--------|
| 특징 추출 | 사람이 직접 설계 | 자동으로 학습 |
| 층의 수 | 1~2개 | 수십~수백 개 |
| 데이터 양 | 적은 양 가능 | 대량 필요 |
| 컴퓨팅 | 일반 PC 가능 | GPU 필요 |

## 📚 학습 과정: 역전파

딥러닝은 **역전파(Backpropagation)**라는 방법으로 학습해요:

1. **순전파**: 데이터를 넣고 예측값을 구해요
2. **오차 계산**: 예측값과 정답의 차이(오차)를 계산해요
3. **역전파**: 오차를 뒤에서 앞으로 전달하며 가중치를 조정해요
4. **반복**: 수천~수만 번 반복하며 점점 정확해져요

마치 활쏘기를 연습하는 것과 비슷해요 — 쏘고, 과녁에서 얼마나 벗어났는지 확인하고, 다음번에 조정하는 거예요!

## 🎨 딥러닝의 주요 모델

### CNN (합성곱 신경망)
이미지 처리에 특히 강해요. 사진에서 윤곽, 질감, 형태를 단계적으로 인식해요.

- 얼굴 인식, 자율주행 차량의 시각 처리, 의료 영상 분석

### RNN (순환 신경망)
시계열 데이터와 텍스트 처리에 강해요. 이전 정보를 기억하며 순서를 이해해요.

- 음성 인식, 주가 예측, 번역

### Transformer
자연어 처리의 혁명을 일으킨 모델이에요. 문장 전체를 한 번에 이해해요.

- ChatGPT, Claude, 구글 번역 등 대부분의 최신 AI가 사용

## 💡 딥러닝의 돌파구

딥러닝이 2010년대부터 폭발적으로 성장한 세 가지 이유가 있어요:

1. **빅데이터** — 인터넷 덕분에 학습 데이터가 폭발적으로 늘었어요
2. **GPU** — 그래픽카드의 병렬 처리 능력이 학습 속도를 획기적으로 높였어요
3. **알고리즘 혁신** — Dropout, BatchNorm 등 학습을 안정시키는 기법이 등장했어요

## ⚡ 실습으로 느껴보기

딥러닝을 직접 체험해 보고 싶다면 이런 도구를 활용해 보세요:

## 💬 시나리오 예시

### 예시 1: 의료 영상 진단

**상황**: 방사선과 전문의가 하루 200장의 X-ray를 판독해야 한다.

> ❌ **Before (수동 판독)**
> 의사가 모든 영상을 직접 확인한다. 피로 누적으로 미세한 병변을 놓칠 수 있다.

> ✅ **After (CNN 보조 진단)**
> CNN 모델이 수만 장의 영상을 학습해 의심 부위를 표시한다. 의사는 AI가 표시한 영역을 우선 확인해 정확도와 속도가 모두 향상된다.

### 예시 2: 음성 비서

**상황**: 스마트 스피커가 사용자의 말을 이해해야 한다.

> ❌ **Before (키워드 매칭)**
> "음악 틀어"는 인식하지만, "뭔가 신나는 거 하나 틀어줘"는 이해하지 못한다.

> ✅ **After (딥러닝 음성 인식)**
> RNN/Transformer 기반 모델이 문맥과 뉘앙스를 파악해 다양한 표현을 이해하고 적절한 음악을 재생한다.

### 예시 3: 자율주행 시각 처리

**상황**: 자동차가 전방 카메라 영상에서 보행자·차선·신호를 인식해야 한다.

> ❌ **Before (규칙 기반 영상 처리)**
> "빨간색 원형 = 정지 신호" 같은 규칙을 수작업으로 정의한다. 날씨·조명 변화에 취약하다.

> ✅ **After (딥러닝 객체 인식)**
> 수백만 장의 도로 영상을 학습한 CNN이 비·안개·야간에도 보행자와 신호를 실시간 인식한다.

## 📖 핵심 용어

| 용어 | 설명 | 예시 |
|------|------|------|
| **딥러닝** | 다층 신경망으로 복잡한 패턴을 학습하는 기술 | 이미지 인식, 번역 |
| **인공 신경망** | 뇌의 뉴런 구조를 모방한 연산 모델 | 입력층→은닉층→출력층 |
| **역전파** | 오차를 뒤에서 앞으로 전파해 가중치를 조정하는 학습법 | 활쏘기 후 과녁 확인→조준 조정 |
| **CNN** | 이미지 특징을 단계적으로 추출하는 신경망 | 얼굴 인식, 의료 영상 |
| **RNN** | 순서/시간 정보를 기억하며 처리하는 신경망 | 음성 인식, 주가 예측 |
| **Transformer** | 문장 전체를 병렬로 이해하는 최신 구조 | GPT, Claude, BERT |
| **GPU** | 대량 병렬 연산에 특화된 프로세서 | NVIDIA A100, H100 |

## 📌 핵심 정리

- 딥러닝은 **여러 층의 신경망**으로 복잡한 패턴을 학습해요
- **역전파**를 통해 오차를 줄이며 점점 정확해져요
- CNN(이미지), RNN(시계열), Transformer(언어)가 대표적인 모델이에요
- 빅데이터 + GPU + 알고리즘 혁신이 딥러닝의 성장을 이끌었어요
