# 🔧 Claude Code란?

Claude Code는 Anthropic이 만든 공식 AI 코딩 도구예요. 터미널에서 직접 동작하며, 코드를 읽고, 수정하고, 실행할 수 있는 **에이전틱 코딩 어시스턴트**예요.

## 🆚 기존 AI 코딩 도구와의 차이

| 기능 | 일반 AI 챗봇 | 코드 자동완성 | Claude Code |
|------|-------------|-------------|-------------|
| 코드 제안 | ✅ | ✅ | ✅ |
| 파일 직접 수정 | ❌ | ❌ | ✅ |
| 터미널 명령 실행 | ❌ | ❌ | ✅ |
| 프로젝트 전체 이해 | ❌ | 제한적 | ✅ |
| 멀티파일 작업 | ❌ | ❌ | ✅ |

핵심 차이는 Claude Code가 **실제로 행동할 수 있다**는 점이에요.

## 🚀 시작하기

### 설치
```bash
npm install -g @anthropic-ai/claude-code
```

### 실행
프로젝트 디렉토리에서 실행해요:
```bash
cd my-project
claude
```

### 첫 대화
```
> 이 프로젝트의 구조를 설명해 줘

Claude Code가:
1. 프로젝트 파일을 스캔
2. package.json, 설정 파일 분석
3. 디렉토리 구조와 기술 스택 파악
4. 이해하기 쉽게 정리해서 설명
```

## 💡 무엇을 할 수 있나요?

### 코드 작성
```
> 사용자 로그인 API를 Express.js로 만들어 줘.
> JWT 인증을 사용하고, 입력 검증도 추가해 줘.
```
Claude Code가 파일을 생성하고, 관련 패키지를 설치하고, 테스트까지 작성해요.

### 버그 수정
```
> npm test 실행하면 auth.test.ts에서 실패해.
> 원인 찾아서 고쳐줘.
```
테스트를 실행하고, 에러를 분석하고, 코드를 수정한 뒤, 다시 테스트해서 통과하는지 확인해요.

### 리팩토링
```
> src/utils 폴더의 함수들을 TypeScript로 마이그레이션해 줘.
> 기존 동작은 유지하면서 타입을 추가해 줘.
```

### 코드 리뷰
```
> 최근 커밋들을 리뷰해 줘.
> 성능 문제나 보안 취약점이 있는지 확인해 줘.
```

### Git 작업
```
> 변경사항을 커밋하고, PR을 만들어 줘.
> 커밋 메시지는 Conventional Commits 형식으로 해줘.
```

## 🎮 핵심 워크플로우

### 1. 탐색 → 이해 → 수정
```
"이 프로젝트에서 인증 관련 코드를 찾아서 분석해 줘"
→ Grep, Glob으로 파일 검색
→ Read로 코드 분석
→ 구조와 흐름을 설명

"여기에 2FA 기능을 추가해 줘"
→ 관련 파일 수정
→ 새 파일 생성
→ 테스트 추가
```

### 2. 에러 → 진단 → 해결
```
"npm run build 하면 에러가 나"
→ 빌드 명령 실행
→ 에러 메시지 분석
→ 원인 파악 및 수정
→ 다시 빌드해서 확인
```

## 🛡️ 안전장치

Claude Code는 안전을 위한 장치가 내장되어 있어요:

- **파일 수정 전 확인**: 파일을 바꾸기 전에 항상 사용자에게 물어봐요
- **셸 명령 승인**: 터미널 명령을 실행하기 전에 확인을 받아요
- **변경 내역 추적**: 모든 수정사항을 git으로 추적할 수 있어요
- **되돌리기 가능**: 잘못된 변경은 git으로 쉽게 되돌릴 수 있어요

## ⚡ 생산성 팁

1. **CLAUDE.md 활용**: 프로젝트 규칙을 파일에 적어두면 매번 설명할 필요 없어요
2. **구체적으로 요청**: "이 코드 좀 고쳐줘"보다 "이 함수의 에러 처리를 추가해 줘"가 좋아요
3. **단계별 작업**: 큰 작업은 작은 단계로 나눠서 진행하세요
4. **검증 요청**: "수정한 뒤 테스트 돌려서 확인해 줘"로 품질을 보장하세요
