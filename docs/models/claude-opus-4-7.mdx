## Claude Opus 4.7이란?

Claude Opus 4.7은 Anthropic이 2026년 4월 16일에 출시한 최신 플래그십 모델이에요. 이전 모델인 Opus 4.6의 직접적인 업그레이드 버전으로, 특히 **고난도 소프트웨어 엔지니어링**, **에이전트 워크플로우**, **비전(이미지 처리)**, **멀티스텝 작업**에서 크게 향상됐어요.

:::tip 한 줄 요약
"이전에는 사람이 밀착 감독해야 했던 어려운 코딩 작업을 Opus 4.7에게 자신 있게 맡길 수 있게 됐다" — Anthropic 공식 블로그
:::

## 핵심 특징

| 항목 | 내용 |
|------|------|
| **출시일** | 2026년 4월 16일 |
| **API 모델명** | `claude-opus-4-7` |
| **가격** | 입력 $5/M tokens, 출력 $25/M tokens (Opus 4.6과 동일) |
| **제공 플랫폼** | Claude API, Amazon Bedrock, Google Cloud Vertex AI, Microsoft Foundry |
| **컨텍스트 윈도우** | 기존 Opus 시리즈와 동일한 대규모 컨텍스트 지원 |
| **최상위 모델** | Claude Mythos Preview가 더 강력하나 제한 공개 중 |

## 주요 개선사항

### 코딩 & 에이전트 성능

Opus 4.7의 가장 눈에 띄는 개선은 **자율적인 코딩 능력**이에요.

- 복잡한 장시간 작업을 **일관성 있게** 처리
- 지시사항을 **정확히** 따르고, 결과를 보고하기 전에 **자체 검증** 수행
- Cursor 벤치마크에서 Opus 4.6 대비 **58% → 70%** 달성 (CursorBench)
- 93개 코딩 태스크 벤치마크에서 Opus 4.6 대비 **13% 향상** (Windsurf)
- Opus 4.6과 Sonnet 4.6 모두 풀지 못한 4개 태스크를 추가로 해결

:::info 지시 따르기의 변화
Opus 4.7은 지시사항을 **문자 그대로** 따르는 성향이 강해졌어요. 이전 모델에서는 느슨하게 해석하거나 일부를 건너뛰던 프롬프트가 이제는 정확히 실행되므로, 기존 프롬프트를 재조정할 필요가 있을 수 있어요.
:::

### 향상된 비전 (멀티모달)

- 이미지 최대 해상도가 **2,576px** (긴 변 기준, ~375만 픽셀)로 확대
- 이전 모델 대비 **3배 이상** 높은 해상도 지원
- 화학 구조식, 복잡한 기술 다이어그램, 밀집된 스크린샷 등 세밀한 시각 정보 처리 가능
- XBOW 시각 정확도 벤치마크: Opus 4.6 **54.5%** → Opus 4.7 **98.5%**

### 메모리 & 장기 작업

- 파일 시스템 기반 메모리 활용 능력 향상
- 장기 멀티세션 작업에서 중요한 노트를 기억하고 활용
- 새로운 작업 시 필요한 사전 컨텍스트가 줄어듦
- Devin에서 **수 시간 동안 일관되게** 작업 가능 (이전에는 불가능했던 수준)

### 문서 & 크리에이티브 품질

- 프로페셔널한 인터페이스, 슬라이드, 문서 제작에서 **더 세련된 결과물**
- 금융 분석 보고서, 프레젠테이션 등 실무 문서 품질 대폭 향상
- "대시보드와 데이터 중심 인터페이스를 만드는 데 세계 최고" — Lovable CEO

## 새로운 기능: Effort 레벨

Opus 4.7에서 새로운 `xhigh` (extra high) effort 레벨이 추가됐어요.

| Effort 레벨 | 설명 | 권장 용도 |
|-------------|------|-----------|
| `low` | 빠른 응답, 최소 추론 | 간단한 질문 |
| `medium` | 기본 추론 | 일반 작업 |
| `high` | 깊은 추론 | 복잡한 문제 |
| **`xhigh`** (신규) | high와 max 사이 | 코딩, 에이전트 작업 (Claude Code 기본값) |
| `max` | 최대 추론 | 최고 난도 문제 |

## 토크나이저 변경 사항

Opus 4.7은 **업데이트된 토크나이저**를 사용해요. 같은 입력이 약 **1.0~1.35배** 더 많은 토큰으로 매핑될 수 있어요. 단, 높은 effort 레벨에서 더 깊이 사고하므로 출력 토큰도 증가할 수 있어요.

토큰 사용량을 제어하는 방법:
- effort 파라미터 조정
- task budget 설정 (API 퍼블릭 베타)
- 간결한 응답을 요청하는 프롬프트

## 안전성 & 정렬

Opus 4.7은 Opus 4.6과 유사한 안전성 프로파일을 보여요.

| 항목 | 평가 |
|------|------|
| 기만, 아첨, 오용 협력 | 낮은 비율 (Opus 4.6 수준) |
| 정직성 & 프롬프트 인젝션 방어 | Opus 4.6 대비 **개선** |
| 약물 관련 정보 | Opus 4.6 대비 다소 약화 |
| 전반적 정렬 | "대체로 잘 정렬되고 신뢰할 수 있으나, 완전히 이상적이지는 않음" |
| 사이버보안 | 금지/고위험 사용 자동 탐지 및 차단 세이프가드 탑재 |

:::note Claude Mythos Preview
Mythos Preview가 Anthropic의 가장 잘 정렬된 모델이에요. Opus 4.7은 Mythos의 사이버 역량을 의도적으로 줄여서 학습한 첫 번째 모델로, 실전 배포를 통해 세이프가드를 검증한 뒤 Mythos급 모델의 광범위한 출시를 목표로 하고 있어요.
:::

## Claude Code 업데이트

Opus 4.7 출시와 함께 Claude Code에도 주요 업데이트가 있었어요.

- **기본 effort 레벨**: 모든 플랜에서 `xhigh`로 상향
- **`/ultrareview` 슬래시 커맨드**: 변경사항을 리뷰하고 버그/설계 이슈를 잡아내는 전용 리뷰 세션 (Pro/Max 유저에게 3회 무료 제공)
- **Auto 모드**: Max 유저에게 확대 — Claude가 사용자 대신 결정을 내려 긴 작업을 중단 없이 수행

## 핵심 용어

| 용어 | 설명 |
|------|------|
| **Effort 레벨** | 모델의 추론 깊이를 조절하는 파라미터. 높을수록 더 깊이 사고하지만 응답이 느려짐 |
| **Task Budget** | API에서 Claude의 토큰 사용량을 제어하는 기능 (퍼블릭 베타) |
| **Claude Mythos Preview** | Anthropic의 차세대 최상위 모델. 현재 제한 공개 중 |
| **Project Glasswing** | AI 모델의 사이버보안 위험과 혜택을 다루는 Anthropic 주도 이니셔티브 |
| **Cyber Verification Program** | 합법적 보안 목적으로 Opus 4.7을 사용하려는 전문가를 위한 인증 프로그램 |
