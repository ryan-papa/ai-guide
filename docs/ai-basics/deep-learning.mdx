# 🧠 딥러닝은 어떻게 배우나요?

딥러닝(Deep Learning)은 인간의 뇌를 모방한 **인공 신경망**을 여러 층으로 쌓아, 복잡한 패턴을 학습하는 머신러닝의 한 분야예요.

## 🏭 비유로 이해하기: 딥러닝은 공장 조립 라인

딥러닝을 **자동차 공장의 조립 라인**에 비유해 볼게요.

- **1단계 (입력층)**: 원자재(철판, 유리)가 들어와요 → 이미지의 픽셀 데이터
- **2단계 (은닉층 1)**: 철판을 자르고 모양을 잡아요 → 이미지에서 선과 가장자리 인식
- **3단계 (은닉층 2)**: 부품을 조립해요 → 선들을 조합해 눈, 코, 귀 인식
- **4단계 (은닉층 3)**: 완성차를 검수해요 → 눈+코+귀를 조합해 "고양이 얼굴" 인식
- **5단계 (출력층)**: 출하! → "이것은 고양이입니다" 판정

각 단계가 이전 단계의 결과를 받아 점점 더 복잡하고 추상적인 특징을 만들어내요. 이것이 딥러닝의 핵심이에요.

## 🏗️ 신경망의 구조

인공 신경망은 뇌의 뉴런을 본떠 만들었어요:

```
입력층        은닉층(여러 개)      출력층
[픽셀값]  →  [특징 추출]      →  [결과]
[색상]    →  [패턴 인식]      →  [고양이? 강아지?]
[형태]    →  [고수준 이해]    →
```

- **입력층**: 데이터를 받아들여요 (이미지의 픽셀, 텍스트의 단어 등)
- **은닉층**: 데이터의 특징을 추출하고 변환해요 (이 층이 "깊어서" 딥러닝!)
- **출력층**: 최종 결과를 내놓아요

## 🎯 딥러닝이 "깊은" 이유

머신러닝과 딥러닝의 차이는 **층의 깊이**에 있어요:

| 구분 | 머신러닝 | 딥러닝 |
|------|---------|--------|
| 특징 추출 | 사람이 직접 설계 | 자동으로 학습 |
| 층의 수 | 1~2개 | 수십~수백 개 |
| 데이터 양 | 적은 양 가능 | 대량 필요 |
| 컴퓨팅 | 일반 PC 가능 | GPU 필요 |

## 📚 학습 과정: 역전파

딥러닝은 **역전파(Backpropagation)**라는 방법으로 학습해요:

1. **순전파**: 데이터를 넣고 예측값을 구해요
2. **오차 계산**: 예측값과 정답의 차이(오차)를 계산해요
3. **역전파**: 오차를 뒤에서 앞으로 전달하며 가중치를 조정해요
4. **반복**: 수천~수만 번 반복하며 점점 정확해져요

마치 활쏘기를 연습하는 것과 비슷해요 — 쏘고, 과녁에서 얼마나 벗어났는지 확인하고, 다음번에 조정하는 거예요!

### 코드로 보는 간단한 신경망

PyTorch로 아주 간단한 신경망을 만들면 이런 모습이에요:

```python
import torch
import torch.nn as nn

# 간단한 3층 신경망 정의
class SimpleNet(nn.Module):
    def __init__(self):
        super().__init__()
        self.layer1 = nn.Linear(784, 128)  # 입력층 → 은닉층1
        self.layer2 = nn.Linear(128, 64)   # 은닉층1 → 은닉층2
        self.layer3 = nn.Linear(64, 10)    # 은닉층2 → 출력층 (0~9 숫자)
        self.relu = nn.ReLU()              # 활성화 함수

    def forward(self, x):
        x = self.relu(self.layer1(x))  # 1층 통과 후 활성화
        x = self.relu(self.layer2(x))  # 2층 통과 후 활성화
        x = self.layer3(x)             # 최종 출력
        return x

# 이 모델은 손글씨 숫자(0~9)를 인식할 수 있어요!
model = SimpleNet()
```

`nn.Linear`가 하나의 층이고, 이 층을 여러 개 쌓는 것이 "딥(deep)"러닝이에요. 각 층의 가중치가 역전파를 통해 자동으로 조정돼요.

## 🎨 딥러닝의 주요 모델

### CNN (합성곱 신경망)
이미지 처리에 특히 강해요. 사진에서 윤곽, 질감, 형태를 단계적으로 인식해요.

- 얼굴 인식, 자율주행 차량의 시각 처리, 의료 영상 분석

> 💡 **쉬운 비유**: CNN은 돋보기로 사진을 훑는 것과 비슷해요. 작은 영역씩 살펴보면서 "여기는 눈", "여기는 코" 같은 특징을 찾고, 이를 종합해 "이건 고양이"라고 판단해요.

### RNN (순환 신경망)
시계열 데이터와 텍스트 처리에 강해요. 이전 정보를 기억하며 순서를 이해해요.

- 음성 인식, 주가 예측, 번역

> 💡 **쉬운 비유**: RNN은 소설을 읽는 독자와 같아요. 앞에서 읽은 내용을 기억하면서 다음 내용을 이해해요. "그가 문을 열었다"에서 "그"가 누구인지 앞 문맥으로 파악하는 거예요.

### Transformer
자연어 처리의 혁명을 일으킨 모델이에요. 문장 전체를 한 번에 이해해요.

- ChatGPT, Claude, 구글 번역 등 대부분의 최신 AI가 사용

> 💡 **쉬운 비유**: Transformer는 시험지를 받고 전체를 한눈에 훑어보는 학생과 같아요. RNN이 한 문제씩 순서대로 푸는 방식이라면, Transformer는 모든 문제를 동시에 보고 어떤 문제가 서로 관련 있는지 파악해요. 이 "주목(Attention)" 메커니즘 덕분에 훨씬 빠르고 정확해요.

## 💡 딥러닝의 돌파구

딥러닝이 2010년대부터 폭발적으로 성장한 세 가지 이유가 있어요:

1. **빅데이터** — 인터넷 덕분에 학습 데이터가 폭발적으로 늘었어요
2. **GPU** — 그래픽카드의 병렬 처리 능력이 학습 속도를 획기적으로 높였어요
3. **알고리즘 혁신** — Dropout, BatchNorm 등 학습을 안정시키는 기법이 등장했어요

## 💬 시나리오 예시

### 예시 1: 의료 영상 진단

**상황**: 방사선과 전문의가 하루 200장의 X-ray를 판독해야 한다.

> ❌ **Before (수동 판독)**
> 의사가 모든 영상을 직접 확인한다. 피로 누적으로 미세한 병변을 놓칠 수 있다.

> ✅ **After (CNN 보조 진단)**
> CNN 모델이 수만 장의 영상을 학습해 의심 부위를 표시한다. 의사는 AI가 표시한 영역을 우선 확인해 정확도와 속도가 모두 향상된다.

### 예시 2: 음성 비서

**상황**: 스마트 스피커가 사용자의 말을 이해해야 한다.

> ❌ **Before (키워드 매칭)**
> "음악 틀어"는 인식하지만, "뭔가 신나는 거 하나 틀어줘"는 이해하지 못한다.

> ✅ **After (딥러닝 음성 인식)**
> RNN/Transformer 기반 모델이 문맥과 뉘앙스를 파악해 다양한 표현을 이해하고 적절한 음악을 재생한다.

### 예시 3: 자율주행 시각 처리

**상황**: 자동차가 전방 카메라 영상에서 보행자·차선·신호를 인식해야 한다.

> ❌ **Before (규칙 기반 영상 처리)**
> "빨간색 원형 = 정지 신호" 같은 규칙을 수작업으로 정의한다. 날씨·조명 변화에 취약하다.

> ✅ **After (딥러닝 객체 인식)**
> 수백만 장의 도로 영상을 학습한 CNN이 비·안개·야간에도 보행자와 신호를 실시간 인식한다.

## 📖 핵심 용어

| 용어 | 설명 | 예시 |
|------|------|------|
| **딥러닝** | 다층 신경망으로 복잡한 패턴을 학습하는 기술 | 이미지 인식, 번역 |
| **인공 신경망** | 뇌의 뉴런 구조를 모방한 연산 모델 | 입력층→은닉층→출력층 |
| **역전파** | 오차를 뒤에서 앞으로 전파해 가중치를 조정하는 학습법 | 활쏘기 후 과녁 확인→조준 조정 |
| **CNN** | 이미지 특징을 단계적으로 추출하는 신경망 | 얼굴 인식, 의료 영상 |
| **RNN** | 순서/시간 정보를 기억하며 처리하는 신경망 | 음성 인식, 주가 예측 |
| **Transformer** | 문장 전체를 병렬로 이해하는 최신 구조 | GPT, Claude, BERT |
| **GPU** | 대량 병렬 연산에 특화된 프로세서 | NVIDIA A100, H100 |
| **가중치(Weight)** | 신경망 연결의 강도를 나타내는 숫자, 학습을 통해 조정됨 | 각 뉴런 연결의 중요도 |
| **활성화 함수** | 뉴런의 출력값을 변환하는 함수 (비선형성 부여) | ReLU, Sigmoid, Tanh |
| **에포크(Epoch)** | 전체 학습 데이터를 한 번 다 보는 것 | 100에포크 = 전체 데이터 100번 반복 |
| **배치(Batch)** | 한 번에 학습하는 데이터 묶음의 크기 | 배치 크기 32 = 32개씩 묶어 학습 |

## 🆚 머신러닝 vs 딥러닝: 언제 무엇을 쓸까?

| 상황 | 추천 | 이유 |
|------|------|------|
| 데이터가 적다 (수백~수천 건) | 머신러닝 | 딥러닝은 대량 데이터가 필요해요 |
| 정형 데이터 (표, 수치) | 머신러닝 | XGBoost 등이 더 효율적이에요 |
| 이미지·음성·텍스트 | 딥러닝 | 비정형 데이터는 딥러닝이 압도적이에요 |
| 결과 해석이 중요하다 | 머신러닝 | 의사결정트리 등은 해석이 쉬워요 |
| 최고 성능이 필요하다 | 딥러닝 | 데이터와 컴퓨팅이 충분하다면 딥러닝이 유리해요 |

## 📌 핵심 정리

- 딥러닝은 **여러 층의 신경망**으로 복잡한 패턴을 학습해요
- **역전파**를 통해 오차를 줄이며 점점 정확해져요
- CNN(이미지), RNN(시계열), Transformer(언어)가 대표적인 모델이에요
- 빅데이터 + GPU + 알고리즘 혁신이 딥러닝의 성장을 이끌었어요
