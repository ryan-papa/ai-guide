# 🤖 AI란 무엇인가요?

인공지능(Artificial Intelligence, AI)은 인간의 지능을 모방하여 학습하고, 추론하고, 문제를 해결할 수 있는 컴퓨터 시스템을 말해요.

## 🍳 비유로 이해하기: AI는 요리사

AI를 이해하는 가장 쉬운 방법은 **요리사**에 비유하는 거예요.

- **전통 프로그래밍** = 레시피를 그대로 따라하는 초보 요리사. "소금 5g, 물 200ml"처럼 정해진 절차만 수행해요.
- **AI** = 경험이 풍부한 셰프. 수천 가지 요리를 맛본 경험을 바탕으로, 새로운 재료를 봐도 어떤 요리를 만들면 좋을지 **스스로 판단**할 수 있어요.

즉, AI의 핵심은 **"미리 정해진 규칙 없이도, 경험(데이터)으로부터 스스로 배우는 것"**이에요.

## 💡 AI의 핵심 능력

AI는 크게 네 가지 능력을 갖추고 있어요:

| 능력 | 설명 | 예시 |
|------|------|------|
| **인식** | 이미지, 음성, 텍스트를 이해해요 | 얼굴 인식, 음성 인식 |
| **학습** | 데이터에서 패턴을 찾아요 | 스팸 메일 분류 |
| **추론** | 논리적 판단을 내려요 | 의료 진단 보조 |
| **생성** | 새로운 콘텐츠를 만들어요 | 글쓰기, 그림 그리기 |

## 🏠 일상 속 AI

우리는 이미 매일 AI를 사용하고 있어요:

- **음성 비서** — 시리, 빅스비, 알렉사에게 날씨를 물어보는 것
- **추천 시스템** — 넷플릭스가 "이 영화 좋아할 걸요?" 하는 것
- **번역** — 파파고, 구글 번역으로 외국어를 이해하는 것
- **자동완성** — 이메일이나 검색어를 미리 예측하는 것
- **네비게이션** — 실시간 교통 상황을 분석해 최적 경로를 안내하는 것

## 🎯 AI의 세 가지 수준

AI는 능력에 따라 세 단계로 나눌 수 있어요:

### 1. 약한 AI (Narrow AI)
특정 작업 하나를 잘 수행하는 AI예요. 현재 우리가 사용하는 대부분의 AI가 여기에 해당해요.
- 체스를 두는 AI, 번역하는 AI, 얼굴을 인식하는 AI

### 2. 일반 AI (AGI)
인간처럼 다양한 분야에서 범용적으로 사고할 수 있는 AI예요. 아직 완전히 실현되지는 않았지만, 최신 LLM들이 코딩·추론·분석 등 다양한 분야에서 전문가 수준을 보이면서 AGI에 점점 가까워지고 있다는 평가를 받고 있어요.

### 3. 초인공지능 (ASI)
인간의 지능을 뛰어넘는 가상의 AI예요. 현재로서는 이론적 개념이에요.

## 🔑 AI vs 전통 프로그래밍

전통적인 프로그래밍과 AI의 가장 큰 차이점은 **학습**이에요:

```
전통 프로그래밍: 규칙을 사람이 직접 작성
  입력 + 규칙 → 출력

AI/머신러닝: 데이터로부터 규칙을 학습
  입력 + 출력 → 규칙 (자동 학습)
```

예를 들어 "고양이 사진을 판별하는 프로그램"을 만든다고 해볼게요:
- **전통 방식**: "귀가 뾰족하고, 수염이 있고..." 하고 규칙을 일일이 작성해요
- **AI 방식**: 수천 장의 고양이 사진을 보여주고, AI가 스스로 특징을 학습해요

이를 Python 수도코드로 표현하면 이런 차이예요:

```python
# 전통 프로그래밍 방식
def is_cat_traditional(image):
    if has_pointy_ears(image) and has_whiskers(image) and has_fur(image):
        return True  # 규칙을 사람이 하나하나 정의
    return False

# AI 방식 (머신러닝)
def is_cat_ai(image):
    model = train(cat_photos_10000장, labels)  # 데이터로 학습
    return model.predict(image)  # 스스로 배운 규칙으로 판별
```

## 💬 시나리오 예시

### 예시 1: 이메일 스팸 필터

**상황**: 매일 100통의 이메일 중 스팸을 걸러내야 한다.

> ❌ **Before (규칙 기반)**
> "제목에 '광고'가 포함되면 스팸으로 분류" — 규칙을 수백 개 작성해도 새로운 유형의 스팸에 대응하지 못한다.

> ✅ **After (AI 방식)**
> 과거 스팸/정상 이메일을 학습시키면 AI가 패턴을 파악한다. 새로운 유형의 스팸도 유사 패턴으로 감지한다.

### 예시 2: 사진 분류

**상황**: 여행 사진 1만 장에서 가족 사진만 골라내야 한다.

> ❌ **Before (수동)**
> 사진을 하나씩 열어 직접 분류한다. 수일이 걸린다.

> ✅ **After (AI 활용)**
> 얼굴 인식 AI가 가족 얼굴을 학습하고 자동 분류한다. 수분 내 완료된다.

### 예시 3: 고객 문의 응대

**상황**: 쇼핑몰에 하루 500건의 문의가 들어온다.

> ❌ **Before (키워드 챗봇)**
> "배송" 키워드가 있으면 배송 안내를 표시한다. "물건이 안 왔어요"처럼 키워드가 없으면 응답하지 못한다.

> ✅ **After (AI 챗봇)**
> 문맥을 이해해서 "물건이 안 왔어요" = 배송 문의임을 파악하고 적절히 응답한다.

## 📖 핵심 용어

| 용어 | 설명 | 예시 |
|------|------|------|
| **인공지능(AI)** | 인간의 지능을 모방하는 컴퓨터 시스템 | Siri, ChatGPT, Claude |
| **약한 AI(Narrow AI)** | 특정 작업 하나에 특화된 AI | 체스 AI, 번역기 |
| **일반 AI(AGI)** | 인간처럼 범용적으로 사고하는 AI | 아직 미실현 |
| **알고리즘** | 문제를 해결하기 위한 단계별 절차 | 정렬, 검색, 추천 알고리즘 |
| **학습 데이터** | AI가 패턴을 배우는 데 사용하는 데이터 | 고양이 사진 1만 장 |
| **패턴 인식** | 데이터에서 규칙성을 찾아내는 능력 | 스팸 메일의 공통 특징 발견 |
| **추론(Inference)** | 학습된 모델이 새로운 데이터에 대해 판단을 내리는 과정 | 처음 보는 사진이 고양이인지 판별 |
| **생성형 AI(Generative AI)** | 새로운 텍스트, 이미지, 음악 등을 만들어내는 AI | ChatGPT, DALL·E, Midjourney |

| 산업 | 활용 사례 | 대표 서비스 |
|------|----------|------------|

| **금융** | 이상 거래 탐지, 신용 평가 | PayPal 사기 탐지 |
| **교통** | 자율주행, 교통 흐름 최적화 | Tesla Autopilot, Waymo |
| **교육** | 맞춤형 학습, 자동 채점 | Duolingo, Khan Academy |
| **엔터테인먼트** | 콘텐츠 추천, AI 음악/그림 생성 | Netflix, Spotify, Midjourney |

## 📌 핵심 정리

- AI는 **인간의 지능을 모방**하는 컴퓨터 시스템이에요
- 현재 AI는 대부분 **특정 작업에 특화된 약한 AI**예요
- AI의 핵심은 **데이터로부터 스스로 학습**하는 것이에요
- 우리는 이미 일상에서 AI를 널리 활용하고 있어요
