## 개요·목적

- **배경:** AI 기술이 빠르게 발전하면서 초보자부터 실무자까지 체계적으로 학습할 수 있는 한국어 가이드가 부족하다. 특히 Claude Code 도구 활용법을 포함한 실무 중심 콘텐츠가 필요하다.
- **목표:** AI 기본 개념부터 Claude Code 도구까지 4개 카테고리로 단계별 학습할 수 있는 정적 웹사이트를 구축한다.
- **범위:** 콘텐츠 4개 카테고리 (AI 기초 / AI 발전사 / 엔지니어링 기법 / Claude Code 도구), 정적 사이트 빌드·배포, 검색·네비게이션·다크모드 기능

## 사용자 스토리

| As a | I want | So that |
|------|--------|---------|
| AI 입문자 | 기초 개념(ML, DL, LLM)을 단계별로 읽고 싶다 | AI 전반을 체계적으로 이해할 수 있다 |
| 개발자 | 프롬프트·컨텍스트·에이전틱 엔지니어링 기법을 학습하고 싶다 | 실무에 바로 적용할 수 있다 |
| Claude Code 사용자 | 스킬, 플러그인, MCP 등 도구 활용법을 찾고 싶다 | 개발 생산성을 높일 수 있다 |
| 모든 사용자 | 난이도 태그와 검색으로 원하는 콘텐츠를 빠르게 찾고 싶다 | 학습 시간을 절약할 수 있다 |
