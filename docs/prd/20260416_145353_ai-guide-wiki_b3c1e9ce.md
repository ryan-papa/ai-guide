## 개요·목적

- **배경:** 기존 AI Guide는 Astro 기반 정적 사이트로, 고정 4개 카테고리(AI 기초/발전사/엔지니어링/Claude Code)에 16개 글이 있다. 사용자가 문서/링크를 전달하면 AI가 분석·가공해 콘텐츠를 지속 축적하는 **종합 AI/ML 위키**로 전환이 필요하다.
- **목표:** Docusaurus 기반 AI/ML 종합 위키를 구축한다. 텔레그램으로 문서/링크를 전달하면 분석→카테고리화→MDX 문서 생성→PR까지 자동화한다.
- **범위:** 프레임워크 전환(Astro→Docusaurus), 카테고리 체계 재설계, 기존 콘텐츠 마이그레이션, 토스 스타일 테마 커스터마이징, GitHub Pages 배포

## 사용자 스토리

| As a | I want | So that |
|------|--------|---------|
| AI 입문자 | 기초 개념을 예시와 함께 단계별로 학습하고 싶다 | AI 전반을 체계적으로 이해할 수 있다 |
| 개발자 | 엔지니어링 기법/도구의 실무 활용법을 찾고 싶다 | 개발에 바로 적용할 수 있다 |
| 위키 운영자 | 텔레그램에 링크만 던지면 위키에 문서가 추가되길 원한다 | 콘텐츠 축적 비용이 최소화된다 |
| 모든 사용자 | 검색과 카테고리로 원하는 정보를 빠르게 찾고 싶다 | 학습 시간을 절약할 수 있다 |

## 디자인 원칙

- 토스 스타일 미니멀: 흰 배경(`#FFFFFF`) + 블루 포인트(`#3182F6`)
- 다크모드: 시스템 설정 감지 + 수동 토글, 어두운 배경(`#191F28`)
- 타이포그래피: 가독성 우선, 충분한 행간
- 콘텐츠 문체: 간결하되 예시와 배경지식을 풍부하게 제공, 초보자 친화
