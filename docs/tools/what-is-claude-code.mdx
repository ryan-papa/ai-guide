## Claude Code란?

Claude Code는 Anthropic이 만든 공식 AI 코딩 도구예요. 터미널에서 직접 동작하며, 코드를 읽고, 수정하고, 실행할 수 있는 **에이전틱 코딩 어시스턴트**예요.

:::tip 쉬운 비유
Claude Code는 **터미널에 상주하는 시니어 개발자**라고 생각하면 돼요. 일반 AI 챗봇이 "이렇게 고치세요"라고 *조언*만 하는 상담사라면, Claude Code는 직접 키보드를 잡고 코드를 수정하고, 빌드를 돌리고, 테스트까지 확인해 주는 **페어 프로그래밍 파트너**예요.
:::

### 기존 AI 코딩 도구와의 차이

| 기능 | 일반 AI 챗봇 | 코드 자동완성 | Claude Code |
|------|-------------|-------------|-------------|
| 코드 제안 | ✅ | ✅ | ✅ |
| 파일 직접 수정 | ❌ | ❌ | ✅ |
| 터미널 명령 실행 | ❌ | ❌ | ✅ |
| 프로젝트 전체 이해 | ❌ | 제한적 | ✅ |
| 멀티파일 작업 | ❌ | ❌ | ✅ |

핵심 차이는 Claude Code가 **실제로 행동할 수 있다**는 점이에요.

### 시작하기

#### 설치
```bash
npm install -g @anthropic-ai/claude-code
```

#### 실행
프로젝트 디렉토리에서 실행해요:
```bash
cd my-project
claude
```

#### 첫 대화
```
> 이 프로젝트의 구조를 설명해 줘

Claude Code가:
1. 프로젝트 파일을 스캔
2. package.json, 설정 파일 분석
3. 디렉토리 구조와 기술 스택 파악
4. 이해하기 쉽게 정리해서 설명
```

### 무엇을 할 수 있나요?

#### 코드 작성
```
> 사용자 로그인 API를 Express.js로 만들어 줘.
> JWT 인증을 사용하고, 입력 검증도 추가해 줘.
```
Claude Code가 파일을 생성하고, 관련 패키지를 설치하고, 테스트까지 작성해요.

#### 버그 수정
```
> npm test 실행하면 auth.test.ts에서 실패해.
> 원인 찾아서 고쳐줘.
```
테스트를 실행하고, 에러를 분석하고, 코드를 수정한 뒤, 다시 테스트해서 통과하는지 확인해요.

#### 리팩토링
```
> src/utils 폴더의 함수들을 TypeScript로 마이그레이션해 줘.
> 기존 동작은 유지하면서 타입을 추가해 줘.
```

#### 코드 리뷰
```
> 최근 커밋들을 리뷰해 줘.
> 성능 문제나 보안 취약점이 있는지 확인해 줘.
```

#### Git 작업
```
> 변경사항을 커밋하고, PR을 만들어 줘.
> 커밋 메시지는 Conventional Commits 형식으로 해줘.
```

### 핵심 워크플로우

#### 1. 탐색 → 이해 → 수정
```
"이 프로젝트에서 인증 관련 코드를 찾아서 분석해 줘"
→ Grep, Glob으로 파일 검색
→ Read로 코드 분석
→ 구조와 흐름을 설명

"여기에 2FA 기능을 추가해 줘"
→ 관련 파일 수정
→ 새 파일 생성
→ 테스트 추가
```

#### 2. 에러 → 진단 → 해결
```
"npm run build 하면 에러가 나"
→ 빌드 명령 실행
→ 에러 메시지 분석
→ 원인 파악 및 수정
→ 다시 빌드해서 확인
```

### 안전장치

Claude Code는 안전을 위한 장치가 내장되어 있어요:

- **파일 수정 전 확인**: 파일을 바꾸기 전에 항상 사용자에게 물어봐요
- **셸 명령 승인**: 터미널 명령을 실행하기 전에 확인을 받아요
- **변경 내역 추적**: 모든 수정사항을 git으로 추적할 수 있어요
- **되돌리기 가능**: 잘못된 변경은 git으로 쉽게 되돌릴 수 있어요

### 생산성 팁

1. **CLAUDE.md 활용**: 프로젝트 규칙을 파일에 적어두면 매번 설명할 필요 없어요
2. **구체적으로 요청**: "이 코드 좀 고쳐줘"보다 "이 함수의 에러 처리를 추가해 줘"가 좋아요
3. **단계별 작업**: 큰 작업은 작은 단계로 나눠서 진행하세요
4. **검증 요청**: "수정한 뒤 테스트 돌려서 확인해 줘"로 품질을 보장하세요

#### CLAUDE.md 실제 작성 예시

프로젝트 루트에 `CLAUDE.md` 파일을 만들어 두면 Claude Code가 매번 자동으로 읽어요:

```markdown
# 프로젝트 규칙

## 기술 스택
- Framework: Next.js 14 (App Router)
- Language: TypeScript (strict mode)
- 스타일: Tailwind CSS

## 코딩 컨벤션
- 함수명: camelCase
- 컴포넌트명: PascalCase
- 커밋 메시지: Conventional Commits (feat:, fix:, docs: 등)

## 중요 규칙
- console.log 대신 logger 유틸 사용
- API 응답은 반드시 try-catch로 감싸기
- 새 함수 작성 시 단위 테스트 필수
```

:::info CLAUDE.md는 어디에?
- **프로젝트 루트**: 해당 프로젝트에만 적용
- **홈 디렉토리** (`~/.claude/CLAUDE.md`): 모든 프로젝트에 공통 적용
- 프로젝트 설정이 글로벌 설정보다 우선해요
:::

#### 자주 쓰는 명령어 모음

```bash
# Claude Code 시작 (대화형)
claude

# 한 줄 명령으로 바로 실행 (비대화형)
claude -p "이 프로젝트의 README를 한국어로 작성해 줘"

# 대화 이어가기 (이전 컨텍스트 유지)
claude --continue

# 특정 모델 지정
claude --model claude-sonnet-4-20250514

# 비용 확인
claude --cost
```

### 시나리오 예시

#### 예시 1: 새 기능 구현

**상황**: Express.js 프로젝트에 비밀번호 재설정 API를 추가해야 한다.

> ❌ **Before (수동 개발)**
> 구글링으로 예제를 찾고, 복사-붙여넣기로 조합하고, 프로젝트 구조에 맞게 수동 수정한다. 반나절이 걸린다.

> ✅ **After (Claude Code)**
> "비밀번호 재설정 API를 추가해 줘. 이메일 토큰 방식으로."라고 요청하면 프로젝트 구조를 분석 → 라우트·컨트롤러·테스트 생성 → 빌드 확인까지 자동 수행한다.

#### 예시 2: 레거시 마이그레이션

**상황**: JavaScript 파일 30개를 TypeScript로 전환해야 한다.

> ❌ **Before (수동 변환)**
> 파일마다 타입을 하나씩 추가하고, import/export를 수정하고, 빌드 에러를 잡는다. 수일이 걸린다.

> ✅ **After (Claude Code)**
> "src/ 폴더의 JS 파일을 TS로 변환해 줘. 기존 동작 유지하면서 strict 타입 추가해 줘."라고 요청하면 파일별로 변환 → 타입 에러 해결 → 빌드 통과까지 반복 수행한다.

#### 예시 3: 긴급 버그 대응

**상황**: 프로덕션에서 500 에러가 발생했다.

> ❌ **Before (수동 디버깅)**
> 에러 로그 확인 → 관련 코드 탐색 → 원인 추론 → 수정 → 테스트. 각 단계를 직접 수행한다.

> ✅ **After (Claude Code)**
> "npm test 실행하면 auth.test.ts에서 실패해. 원인 찾아서 고쳐줘."라고 요청하면 테스트 실행 → 에러 분석 → 코드 수정 → 재테스트까지 한 번에 완료한다.

### 핵심 용어

| 용어 | 설명 | 예시 |
|------|------|------|
| **Claude Code** | Anthropic의 공식 터미널 기반 AI 코딩 도구 | `claude` 명령으로 실행 |
| **에이전틱 코딩** | AI가 코드를 직접 읽고·수정·실행하는 방식 | 파일 편집 + 셸 명령 + 테스트 |
| **CLAUDE.md** | 프로젝트 규칙을 Claude Code에 전달하는 설정 파일 | 코딩 컨벤션, 아키텍처 규칙 |
| **승인 프롬프트** | 위험한 작업 전 사용자에게 확인을 요청하는 안전장치 | 파일 수정·셸 명령 실행 전 확인 |
| **멀티파일 작업** | 여러 파일을 동시에 읽고 수정하는 능력 | 30개 파일 TypeScript 변환 |
| **CLI** | Command Line Interface, 터미널에서 텍스트로 조작하는 인터페이스 | `claude` 명령어 |
| **페어 프로그래밍** | 두 사람(여기서는 사람+AI)이 함께 코드를 작성하는 방식 | 내가 방향을 제시하고 Claude Code가 구현 |

### 핵심 정리

- Claude Code는 터미널에서 동작하는 **에이전틱 코딩 도구**예요
- 코드를 읽고, 수정하고, 실행하고, 테스트까지 **직접** 할 수 있어요
- **안전장치**가 내장되어 있어 위험한 작업은 사전에 확인해요
- `CLAUDE.md`로 프로젝트 규칙을 설정하면 더 효과적으로 사용할 수 있어요
