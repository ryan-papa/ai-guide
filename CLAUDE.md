# AI Guide 프로젝트 규칙

## 프로젝트 개요

- Docusaurus 3.x 기반 AI/ML 종합 위키
- 한국어 전용, 정적 사이트 (GitHub Pages)
- 라이브: https://ryan-papa.github.io/ai-guide/

## 배포 자동 승인

**문서 추가/수정 작업은 전 단계 자동 진행 (배포 포함, 사용자 승인 불필요).**

| 작업 유형 | 승인 | 비고 |
|----------|:----:|------|
| 문서 추가 (MDX/MD) | 자동 | 배포까지 자동 |
| 문서 수정 (내용·표기·구조) | 자동 | 배포까지 자동 |
| 설정 변경 (config, CSS, 컴포넌트) | 수동 | 공통 규칙 따름 |
| 기능 추가 (플러그인, 신규 컴포넌트) | 수동 | 공통 규칙 따름 |

## 콘텐츠 규칙

| 규칙 | 내용 |
|------|------|
| 제목 레벨 | h2부터 시작 (h1 사용 금지) |
| 이모지 | 섹션 제목 금지, 본문 강조용만 허용 |
| 문체 | 해요체 |
| 용어 | 기술 약어 대문자 (AI, LLM, GPU 등), 고유명사 공식 표기 |
| 언어 | 한국어 친화 (영문 UI 텍스트 한글화) |

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프레임워크 | Docusaurus 3.10 |
| 콘텐츠 | MDX |
| 검색 | @easyops-cn/docusaurus-search-local |
| 배포 | GitHub Pages + GitHub Actions |
| 빌드 명령 | `npm run build` |
| 테스트 | 빌드 통과 = 테스트 통과 (콘텐츠 프로젝트) |
